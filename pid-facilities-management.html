Microsoft 365 Connection

Link your work email and calendar to work orders and inspections
ⓘ
This section is a working interface, not a live connection. Connecting a real Microsoft work account requires an Azure AD app registration and a secure backend to hold the OAuth tokens — that can't run safely inside a browser-only app like this one. What's below is fully built and ready to wire up once that backend exists. In the meantime, Claude for Outlook already does real inbox triage against your actual Microsoft account today — worth using in parallel.
